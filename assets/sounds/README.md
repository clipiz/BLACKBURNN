# Sons d'interface (tactique)

Déposez ici des fichiers audio **libres de droits** avec exactement ces noms :

- `hover.mp3` : survol discret (hover)
- `click.mp3` : clic principal
- `confirm.mp3` : confirmation / activation (optionnel, mais recommandé)

Le code JavaScript charge automatiquement ces chemins :

- `assets/sounds/hover.mp3`
- `assets/sounds/click.mp3`
- `assets/sounds/confirm.mp3`

Si un fichier manque, le site continue de fonctionner sans bloquer l'interface.
